# puppet-zfsonlinux

## Support

* CentOS 6.4 x86_64 (Vagrant only)
* Scientific Linux 6.4 x86_64

## Usage

    class { 'zfsonlinux': }

## Dependencies

* Ruby 1.8.7
* Bundler

## Running tests

1. To install dependencies run `bundle install`
2. Run tests using `rake spec`
3. Verbose test output available using `rake all`
